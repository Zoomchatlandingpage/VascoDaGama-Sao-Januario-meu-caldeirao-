VALORES QUE NÃO SE NEGOCIAM

Este projeto só existe porque alguns valores são inegociáveis.

1. Pertencimento

O Vasco não é produto. É vínculo.
Quem contribui não compra acesso. Reafirma pertencimento.

2. Inclusão

São Januário nunca foi neutro. Sempre foi espaço de quem não tinha espaço.
Modernizar não é excluir. É organizar sem apagar.

3. Comunidade

A Barreira não é problema urbano. É ativo social.
Qualquer futuro que ignore isso, fracassa.

4. Dignidade

O comércio informal não deve ser combatido. Deve ser integrado, qualificado e protegido.
O estádio precisa gerar renda contínua para quem sempre viveu dele.

5. Coragem

O Vasco sempre avançou quando foi contra a corrente. Este projeto segue a mesma lógica.
Não é consenso imediato. É convicção.

Esses valores são o filtro. Qualquer decisão futura que os viole, está fora do projeto.

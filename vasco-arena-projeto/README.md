# Vasco Arena Projeto — São Januário Vivo

Projeto de ativação permanente de São Januário como arena, espaço cultural, comercial e comunitário — construído pela torcida, para a torcida.

## Estrutura do Projeto

| Pasta | Conteúdo |
|-------|----------|
| `00_visao_geral/` | Manifesto, problema atual, visão de futuro e urgência |
| `01_storytelling_e_historia/` | Narrativa emocional, história do estádio, ícones e valores |
| `02_conceito_arena/` | Conceito da arena, diferencial e experiência do torcedor |
| `03_shopping_cruz_de_malta/` | Shopping temático integrado ao estádio |
| `04_modelo_financeiro/` | Meta financeira, tickets, contribuições e fluxo de caixa |
| `05_ticketing_e_socios/` | Níveis de ticket, benefícios e programa de sócios |
| `06_marketing_e_ativacao/` | Fases de marketing, influenciadores e narrativa pública |
| `07_operacional_e_execucao/` | Governança, jurídico, riscos e próximos passos |
| `08_design_e_assets/` | Imagens, renders, logos e referências visuais |
| `09_ai_e_automacoes/` | Uso de IA, prompts, agentes e roadmap técnico |

## Princípio

> "Se em 1927 a torcida levantou paredes, em 2026 ela pode levantar um ecossistema."

O Vasco não começa no apito. Começa na arquibancada.

ORGANIZADAS NÃO SÃO DONAS. SÃO CATALISADORAS.

Erro clássico: entregar o projeto para uma organizada.

Aqui não. Elas entram como canais legítimos de confiança.

Papel real das organizadas

* validação social inicial
* mobilização local
* capilaridade territorial
* leitura da base

Elas não:

* controlam dinheiro
* definem estratégia
* falam sozinhas pelo projeto

O projeto é maior que qualquer grupo.

Por onde começa

* conversas fechadas
* apresentação do manifesto
* escuta real
* ajustes de linguagem
* alinhamento de valores

Quando a organizada fala, o projeto já está redondo.
